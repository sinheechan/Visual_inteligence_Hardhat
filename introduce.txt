영상수집(프레임) -> 분석(Yolov8 detection) -> 시각화 (base64 형태로 받고 즉 str 형태로 받아 용량이 늘어난다.)
영상수집에서는 분석단계로 객체를 보냄
분석에서 시각화는 인코딩해서 보냄

# python predict_api.py

카메라로 객체들을 인식하고 시계, 사람, 헬멧과 같은 객체들을 실시간으로(1초단위) 로 인식한다.
만약 안전지대에 헬멧을 착용한 사람이 입장하게 되면 헬멧 객체를 검출해서 알려준다.
그리고 실제 화면으로 구현하는 것은 프론트앤드 가 하는 역할이로 이러한 알고리즘을 구현하는 것 즉, 이 과정은 백앤드로 넣는 과정이다.


결과가 없을 때 예외처리

# python predict_api.py --weights hardhat.py
